## Setup

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/your-username/wwe-universe-manager.git
    cd wwe-universe-manager
    ```
  </Step>
  <Step title="Install Dependencies">
    ```bash
    npm install
    ```
  </Step>
  <Step title="Setup Environment">
    ```bash
    echo "DATABASE_URL=database.db" > .env
    ```
  </Step>
  <Step title="Setup Database">
    ```bash
    diesel setup
    diesel migration run
    ```
  </Step>
  <Step title="Build CSS">
    ```bash
    npm run build-css-prod
    ```
  </Step>
</Steps>

## Running the Application

### Development Mode (Recommended)

For the best development experience, run these commands in separate terminals:

<CodeGroup>

```bash Terminal 1: CSS Watcher
npm run build-css
```

```bash Terminal 2: Tauri Dev Server
npm run dev
```

</CodeGroup>

The CSS watcher will automatically rebuild styles when you modify Tailwind classes in your Rust components.

### Alternative Development Options

<Tabs>
  <Tab title="Single Command">
    Build CSS once, then run the application:
    ```bash
    npm run build-css-prod && npm run dev
    ```
  </Tab>
  <Tab title="Tauri Direct">
    Use Tauri directly (rebuild CSS manually when needed):
    ```bash
    cargo tauri dev
    ```
  </Tab>
  <Tab title="Frontend Only">
    Frontend-only development without Tauri backend:
    ```bash
    trunk serve
    ```
  </Tab>
</Tabs>

## Production Build

When you're ready to create a production build:

```bash
# Build optimized CSS
npm run build-css-prod

# Build Tauri application
npm run build
```

## Testing

Run the test suite to ensure everything is working:

```bash
# Run all tests
cargo test --workspace

# Run specific package tests
cargo test -p wwe-universe-manager-ui      # Frontend tests
cargo test -p wwe-universe-manager         # Backend tests
```

## First Run Experience

Once the application launches, you'll see the main dashboard. Here's what to do first:

<Steps>
  <Step title="Generate Test Data">
    Click the **"Test Data"** button to populate your database with:
    - 5 sample wrestlers with power ratings
    - 2 wrestling shows (Monday Night RAW, Friday Night SmackDown)
    - 15 championship titles across different divisions
  </Step>
  <Step title="Explore Multi-Window Interface">
    - Click **"Wrestlers"** to open the wrestler management window
    - Click **"Championships"** to open title management
    - Notice how each opens in its own native desktop window
  </Step>
  <Step title="Try Match Booking">
    - Select a show from the dropdown
    - Click **"Book Matches"** to access the Booker Dashboard
    - Create your first match with show roster wrestlers
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Application Won't Start">
    - Ensure all prerequisites are installed
    - Check that CSS was built: `npm run build-css-prod`
    - Verify database setup: `diesel migration run`
    - Try cleaning and rebuilding: `cargo clean && npm run dev`
  </Accordion>
  
  <Accordion title="Database Errors">
    - Check DATABASE_URL in .env file
    - Ensure SQLite3 is installed
    - Run migrations: `diesel migration run`
    - Reset database if needed: `diesel database reset`
  </Accordion>
  
  <Accordion title="CSS Not Loading">
    - Ensure Tailwind CSS is built: `npm run build-css-prod`
    - Check Node.js version (18+ required)
    - Verify npm dependencies: `npm install`
    - Try clearing npm cache: `npm cache clean --force`
  </Accordion>
  
  <Accordion title="Windows Not Opening">
    - Check JavaScript console for errors
    - Ensure application has proper permissions
    - Try restarting the application
    - Check Tauri configuration in `src-tauri/tauri.conf.json`
  </Accordion>
</AccordionGroup>
