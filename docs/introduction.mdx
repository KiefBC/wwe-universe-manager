## What is WWE Universe Manager?

WWE Universe Manager is a comprehensive desktop application built with modern web technologies that allows wrestling fans to create, manage, and organize their own wrestling universe. Take control of shows, wrestlers, championships, and storylines in an immersive WWE-themed environment.

## Technology Stack

WWE Universe Manager is built using cutting-edge technologies:

- **Backend**: Rust with Tauri 2.0 framework
- **Frontend**: Leptos (Rust-based web framework) compiled to WebAssembly
- **Database**: SQLite with Diesel ORM
- **Styling**: Tailwind CSS v4 with DaisyUI components
- **Theme**: Custom WWE-themed interface with red/yellow/black color scheme
